## 目录结构

```
sdp/
├── compute_server_mpi.py      # 主程序（MPI + 4GPU + HIP 融合内核）
├── get_climatology.sh         # 提交脚本（自动选节点）
├── ERA5/Climatology/          # 输出目录（92 个 .nc 文件）
└── .fused_kernel/             # HIP kernel 编译缓存（首次自动生成）
```

## 编译说明

HIP 融合内核通过 PyTorch `cpp_extension.load` 即时编译（JIT），无需手动编译：

- 首次运行：hipcc 编译 4 个 HIP kernel（fused_stats / accumulate_year / finalize / accumulate_batch），约 2-3 分钟
- 编译结果缓存在 `.fused_kernel/` 目录，后续运行秒加载
- 源码 hash 检测：修改 kernel 源码后自动重编译

编译依赖：`ninja` + `setuptools<70`

## 运行命令

```bash
# 默认配置（N_BANDS=12, 自动选空闲节点）
bash get_climatology.sh

# 指定节点
sbatch --nodelist=f16r4n12 get_climatology.sh
```

## 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| N_BANDS | 12 | 纬度分带数（每 GPU 3 bands） |
| IO_THREADS | 2 | 每 rank I/O 线程数 |
| SAVE_WORKERS | 4 | 并行写 nc 进程数 |
| WINDOW_HALF | 5 | 滑动窗口半径（固定） |
| HIP_VISIBLE_DEVICES | 0,1,2,3 | 可见 GPU |
| MPI_NP | 30 | MPI rank 数量 |

## 输出说明

程序在 `ERA5/Climatology/` 目录生成 92 个 NetCDF4 文件，编号 MMDD.nc，对应 6月1日(0601) 至 8月31日(0831)。

每个文件包含 3 个变量：

| 变量 | 维度 | 类型 | 含义 |
|------|------|------|------|
| Climmean | [721×1440] | float32 | 30 年滑动窗口海温气候态均值 |
| P90_sst | [721×1440] | float32 | 30 年滑动窗口 90% 分位阈值 |
| dayofyear | 标量 | int | 年中天序号 (DOY 152-243) |
